## Notes par categorie

### Presse economique (Medias24, L'Economiste, La Vie eco, Challenge, Les Eco)
- Angle principal : tribune d'expert avec donnees exclusives
- Template a utiliser : `template-tribune-expert.md`
- Volume : 1-2 pitches par semaine, alterner les medias
- Objectif : obtenir une tribune publiee avec lien retour vers bati.ma

### BTP specialise (Chantiers du Maroc, Batiactu)
- Angle principal : article de fond sur la digitalisation du secteur
- Template a utiliser : `template-chantiersdumaroc.md` (adapter pour Batiactu)
- Objectif : credibilite sectorielle + backlink depuis un site BTP de reference

### Presse generaliste (Hespress, Le360, TelQuel)
- Angle principal : outil pratique pour les particuliers (calculateur)
- Template a utiliser : `template-communique-presse.md`
- Objectif : visibilite grand public, trafic direct

### Portails immobiliers (Mubawab, Sarouty)
- Angle principal : partenariat de contenu, echange de liens
- Template a utiliser : `template-partenariat-portail.md`
- Objectif : backlink fort + trafic qualifie

### Institutionnel (FNBTP, CNOA)
- Angle principal : outil complementaire, reference sectorielle
- Approche : courrier formel ou rendez-vous en personne
- Objectif : mention officielle, listing dans les ressources, credibilite

### International (Jeune Afrique, Batiactu)
- Angle principal : innovation marocaine, modele exportable au Maghreb/Afrique
- Approche : pitch specifique avec angle panafricain ou comparatif
- Objectif : backlinks internationaux forts, notoriete regionale
